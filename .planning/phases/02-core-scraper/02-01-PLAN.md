<objective>
Build the pure-transform core of the scraper: filter raw marktguru results to the 5-store allow-list, normalize an accepted offer into a frozen StoreOffer (integer cents, cents/litre, Berlin-trimmed dates), select exactly one entry per store under the active-first/upcoming/lowest-price/needsReview ladder, compute deduplicated price-history lines, and expose an injectable clock. Every module is a side-effect-free pure function so the whole pipeline is offline-testable against the captured fixtures.

Purpose: These are the leaf building blocks every downstream module composes ‚Äî they encode DATA-03 normalization, the D-07 selection rule, and the DATA-04 dedup key. They reuse the frozen `contract/matcher.mjs` and `contract/schema.mjs` verbatim; no matching or schema logic is re-implemented here.
Output: `scraper/clock.mjs`, `scraper/normalize.mjs`, `scraper/filter.mjs`, `scraper/select.mjs`, `scraper/dedup.mjs` plus three `node:test` unit files.
</objective>
